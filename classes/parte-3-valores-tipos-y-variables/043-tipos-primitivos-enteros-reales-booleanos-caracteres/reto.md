# Reto de transferencia — Tipos primitivos: enteros, reales, booleanos, caracteres

Añade una cuarta salida `caracter=<c>` con el carácter cuyo código es `n` (por ejemplo 65→'A') y resuélvelo en **Ruby**.

## Criterio de aceptación

- ✅ La salida coincide exactamente con el formato del contrato.
- ✅ Se usa la forma idiomática del lenguaje elegido.
- ✅ Explicas en un comentario qué diferencia (sintáctica/semántica) observaste.
