# 📖 Glosario

> [⬅️ Volver al programa](../README.md) · [📚 Índice completo](../classes/README.md)

Términos recopilados de las secciones *Definiciones y características* de las clases. Cada término
enlaza a la clase donde se explica en contexto, y —cuando aplica— muestra **cómo se llama en cada
lenguaje** (p. ej. *nulo*: `None` en Python, `null` en Java/JS, `nil` en Go, `Option::None` en Rust).

> Este archivo se generará con `python scripts/generar_glosario.py` a partir de las clases
> construidas. No se edita a mano: se corrige la definición en la clase de origen y se regenera.

## Muestra (de la clase 041)

- **Literal** — valor escrito directamente en el código fuente. · [041](../classes/parte-3-valores-tipos-y-variables/041-literales-valores-variables-y-constantes/README.md)
- **Mutabilidad** — si un enlace nombre→valor puede cambiar. · [041](../classes/parte-3-valores-tipos-y-variables/041-literales-valores-variables-y-constantes/README.md)
- **Variable** — nombre asociado a un valor que puede reasignarse. · [041](../classes/parte-3-valores-tipos-y-variables/041-literales-valores-variables-y-constantes/README.md)
